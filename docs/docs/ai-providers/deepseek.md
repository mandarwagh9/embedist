# DeepSeek

DeepSeek offers excellent code capabilities at competitive pricing.

## Setup

1. Go to **Settings → AI Providers**
2. Select **DeepSeek**
3. Enter your API key
4. Select model (default: deepseek-chat)

## Getting an API Key

1. Visit [platform.deepseek.com](https://platform.deepseek.com)
2. Sign up or log in
3. Go to **API Keys**
4. Create a new key

## Models

| Model | Description | Best For |
|-------|-------------|----------|
| deepseek-chat | General chat | Code help |
| deepseek-coder | Code-focused | Programming |

## Pricing

DeepSeek offers competitive pricing. See [DeepSeek Pricing](https://platform.deepseek.com/pricing) for details.
