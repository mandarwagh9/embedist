# Anthropic (Claude)

Claude provides strong code analysis with excellent reasoning capabilities.

## Setup

1. Go to **Settings → AI Providers**
2. Select **Anthropic**
3. Enter your API key
4. Select model (default: claude-sonnet-4-20250514)

## Getting an API Key

1. Visit [console.anthropic.com](https://console.anthropic.com)
2. Sign up or log in
3. Go to **API Keys**
4. Create a new key

## Models

| Model | Description | Best For |
|-------|-------------|----------|
| claude-sonnet-4-20250514 | Latest Sonnet | Balanced |
| claude-opus-4-20250514 | Most capable | Complex tasks |
| claude-3-5-sonnet | Legacy | Budget-conscious |
