# Google Gemini

Gemini provides strong multimodal capabilities and fast response times.

## Setup

1. Go to **Settings → AI Providers**
2. Select **Google Gemini**
3. Enter your API key
4. Select model (default: gemini-2.0-flash)

## Getting an API Key

1. Visit [aistudio.google.com](https://aistudio.google.com)
2. Sign in with Google account
3. Go to **API Keys**
4. Create a new key

## Models

| Model | Description | Best For |
|-------|-------------|----------|
| gemini-2.0-flash | Fast, affordable | Quick queries |
| gemini-2.0-flash-lite | Most efficient | Simple tasks |
| gemini-1.5-pro | Large context | Complex analysis |
