# OpenAI

OpenAI's GPT models provide excellent code understanding and generation capabilities.

## Setup

1. Go to **Settings → AI Providers**
2. Select **OpenAI**
3. Enter your API key
4. Select model (default: gpt-4o)

## Getting an API Key

1. Visit [platform.openai.com](https://platform.openai.com)
2. Sign up or log in
3. Go to **API Keys**
4. Create a new secret key

## Models

| Model | Description | Best For |
|-------|-------------|----------|
| gpt-4o | Latest GPT-4 | Complex debugging |
| gpt-4o-mini | Fast, affordable | Simple queries |
| gpt-4-turbo | Large context | Large codebases |

## Pricing

Pay-per-use pricing based on token usage. See [OpenAI Pricing](https://openai.com/pricing) for details.

## Tips

- Use gpt-4o-mini for quick queries to save costs
- Keep conversations focused for better responses
