# Code Editor

Embedist uses Monaco Editor - the same editor that powers VS Code.

## Features

- **Syntax Highlighting**: C, C++, Arduino (.ino)
- **IntelliSense**: Auto-completion for Arduino/ESP32 APIs
- **Error Highlighting**: Inline compiler error display
- **Multiple Files**: Tab-based editing
- **Find & Replace**: With regex support
- **Minimap**: Code overview

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Save file |
| `Ctrl+N` | New file |
| `Ctrl+O` | Open file |
| `Ctrl+F` | Find |
| `Ctrl+H` | Replace |
| `Ctrl+Shift+F` | Find in files |
| `Ctrl+P` | Quick file open |

## Configuration

Editor settings are in **Settings → Editor**:

- Font size
- Tab size
- Theme (Light/Dark)
- Word wrap
- Minimap visibility

## Arduino Support

The editor includes:
- Arduino function autocomplete
- Library function hints
- Board-specific APIs
- Common error explanations
